# Anatomy of a neural network
```
                              Input X
                                |
                                V
+-----------+       +-------------------------+
|  Weights  |  -->  |          Layer          |
+-----------+       |  (data transformation)  |
                    +-------------------------+
                                |
                                V
+-----------+       +-------------------------+
|  Weights  |  -->  |          Layer          |
+-----------+       |  (data transformation)  |
      ^             +-------------------------+                         
      | update                  |
      | weight                  V
      |                 +---------------+              +----------------+
      |                 |  Predictions  |              |  True targets  |
      |                 |      ^y       |              |       y        |
      |                 +---------------+              +----------------+
      |                         |                             |
      |                         |     +-----------------+     |
      |                         +---->|  Loss function  |<----+
+-------------+                       +-----------------+
|  Optimizer  |                                |
+-------------+                                |
      ^                                        V
      |                                +--------------+      
      +--------------------------------|  Loss score  |
                                       +--------------+
```
